# Ejercicios de POO con TypeScript – Encapsulación, Herencia y Composición

Este proyecto contiene tres actividades prácticas para ejercitar conceptos de Programación Orientada a Objetos en TypeScript:

- **Ejercicio 1:** Encapsulación y abstracción con `UserProfile` y exportadores de perfil.
- **Ejercicio 2:** Herencia y polimorfismo con un sistema de notificaciones.
- **Ejercicio 3:** Composición vs herencia modelando personajes de un juego.

Cada actividad es independiente y tiene su propio archivo de ejemplo.

---

## Requisitos:
- Node.js (versión LTS recomendada)
- npm (se instala junto con Node)
- TypeScript y ts-node (se instalan con npm en este proyecto)

## Instalación:
- Clona el repositorio con:
```bash
git clone https://github.com/ianl98/poo-exercises-mindhub.git
```
- En la raíz del proyecto ejecuta:
```bash
npm install
```

## ¿Cómo ejecutar cada actividad?
- En la raíz del proyecto ejecuta:
```bash
npx ts-node ejercicio1.ts
```

En el caso del ejercicio 2 o el ejercicio 3 solamente se debe cambiar el nombre del ejercicio deseado.
